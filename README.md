# Control Interface for Composite Welding Equipment– postdoctoral project (UNESP/CNPq)

This software was developed as part of a **postdoctoral project** funded by CNPq (Brazil) under call 32/2023 at  
**Universidade Estadual Paulista – UNESP**, aims to provide a professional interface for controlling a composite welding machine using an **Arduino** and a **DPM programmable power supply**.

The interface allows for real-time monitoring of the process, control of voltage and current, data logging, graph generation, and the creation of automated PDF reports.

---

## Technical Features 

- Advanced GUI built with PyQt5 **PyQt5**  
- **Arduino** serial data acquisition (pressure, temperature, and process status)
- **DPM power supply** control via ASCII commands 
- Real-time plotting using **PyQtGraph**  
- **PDF** report generation including final data and charts 
- Automatic data logging to `.txt`  
- Auto-detection of connected serial ports 
- Multithreaded execution for simultaneous communication
- Multi-screen interface utilizing (QStackedWidget)

---

## Technologies Used

- **Python 3.12**
- **PyQt5** – Graphical User Interface (GUI)
- **PySerial** – USB Communication
- **PyQtGraph** – Real-time plotting
- **Matplotlib** – Graphics for PDF generation
- **ReportLab** – PDF document generation
- **Arduino** – Hardware control

## How to Run via Python

In the project folder, execute:

The interface will open immediately.

## Report Generation

At the end of each process, the software automatically:

- Saves a .txt file with all measured values

- Generates a PDF containing:

 - Voltage vs. Time graph
 - User-defined parameters
 - Final measured data
 - Voltage, current, power, pressure, and temperature

##Importing and Exporting Parameters

The interface allows you to:

- Save input parameters to a .txt file
- Load previously saved files.

Ideal for repetitive testing and calibration procedures.
